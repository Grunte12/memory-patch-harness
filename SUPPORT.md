# Support

This project is experimental and maintained as an open-source developer harness.

Use GitHub issues for:

- installation problems,
- schema or validation bugs,
- unclear documentation,
- reproducible evaluation failures,
- adapter requests for a specific agent platform.

Use discussions or a feature request issue for:

- architecture questions,
- proposed retrieval strategies,
- optional integrations such as graph indexes, vector search, compression, or HTML reports.

Please do not post:

- private Obsidian vaults,
- raw agent transcripts containing secrets,
- API keys,
- private customer or workplace data.

When reporting a memory-quality issue, include a small anonymized Memory Patch, Brain Brief, or eval fixture instead of a real vault dump.
